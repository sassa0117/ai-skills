# X（旧Twitter）投稿フォーマット

## 基本ルール
- 140文字以内（厳守）
- フック → 本文 → CTA の3パート構成
- 短ければ短いほどいい（完読率重視）
- 外部リンクはメインツイートに絶対入れない（致命的ペナルティ）
- リンクはセルフリプで対応

## フック（1行目）
最初の1行で「止まらせる」。スクロールを止める力が全て。

### 効くフックパターン
- 意外性: 「〇〇なのに〇〇円」「〇〇円→〇〇万円！！」
- 感嘆: 「エグい」「高すぎる！！」「やばすぎ」
- 自分事化: 「これ家にあったら終わりです」「実家の押し入れ、もう一度だけ見て」
- 速報感: 「〇〇、穴場すぎる」
- 断定: 「マジで穴場」「ガチで知らないと損」

## 本文
- 1-2行で核心を伝える
- 価格差フォーマット: 「○○円→○○円！！」
- 数字は具体的に（「高い」→「13万円」）

## CTA（あれば）
- セルフリプで展開
- 「詳しくはこっちで」→ URL
- メインツイートにはCTA不要（短さ優先）

## テンプレート

### バズ狙い（外部向け・敬語なし）
```
[サイト名]、穴場すぎる

[仕入れ値]円→[販売価格]円！！

[判断理由1文・敬語なし]
```

### フォロワー向け（内輪・敬語あり）
```
[内容]、思ったより[結果]ですね

[詳細]
[学びや気づき・敬語あり]笑
```

### セルフリプ（誘導用）
```
[活動内容]、本気で強化中です

[提供価値]あればこっちで！
[URL]
```

## NG
- 外部リンク直貼り
- 問いかけ形式（「〜じゃない？」→ 断定に変えろ）
- 長文説明（説明はセルフリプで）
- 「知ってた？」系（商材臭い）
- 命令形「〜してください」
- 自己言及メイン（「頑張ってます」系）
