## SECTION 1: ポジショニング & カテゴリーデザイン

### 1-1. ポジショニング理論（Ries & Trout, 1981）[C]
- **核心**: マーケティングの戦場は市場ではなく「顧客の頭の中」。最初にカテゴリーのトップに入ることが最重要
- **応用**: 「○○といえばコレ」と第一想起させるポジションを確保。競合と同じ土俵で戦わず脳内の空きスロットを取る

### 1-2. Obviously Awesome（April Dunford, 2019）[C]
- **核心**: ポジショニングは「競合代替品」から逆算すべき。顧客が自社製品の代わりに何をするかを特定し、そこで自社だけが提供できる差別化価値を明確にする
- **応用**: 10ステッププロセス — 真の競合代替品リスト→独自属性抽出→価値テーママッピング→最適セグメント特定→市場フレーム決定

### 1-3. カテゴリーデザイン（Play Bigger, 2016）[B]
- **核心**: 既存カテゴリーで競争するのではなく、新しいカテゴリーを「設計」して支配する。カテゴリーキングは市場価値の76%を獲得（IPO企業の統計分析）
- **応用**: 「Frame it→Name it→Claim it」の3ステップ。POVで市場を教育し、Lightning Strikeで一気に認知を取る

### 1-4. 22の不変の法則（Ries & Trout, 1993）[D]
- **核心**: マーケティングには破れないルールがある
- **主要法則**:
  - 先行者の法則: 最初であることがベストであることより重要
  - カテゴリーの法則: 1番になれなければ新カテゴリーを作れ
  - 心の法則: 市場で最初より「心の中で最初」が重要
  - 集中の法則: 一つの言葉を所有せよ（ボルボ＝安全）
  - 対立の法則: 2番手はリーダーの逆を行け
  - 延長線の法則: ブランド拡張はブランドを弱める
- **注意**: 後年の研究で一部法則は反証。思考フレームとしては有用

### 1-5. ブルー・オーシャン戦略（Kim & Mauborgne, 2005）[A]
- **核心**: 競争のない新市場空間を創造する。価値イノベーション＝差別化と低コストの同時実現
- **応用**:
  - 戦略キャンバス: 業界の競争要素を可視化し、自社の価値曲線を描く
  - ERRCグリッド: 取り除く/減らす/増やす/付け加える の4アクション
  - 非顧客の3層: まだ顧客でない人を3層に分け最大の市場機会を発見
- **批判**: 「ブルー・オーシャンは事後的にしか特定できない」

### 1-6. ポーターの競争戦略（Porter, 1980）[A]
- **核心**: 競争優位の基本戦略は「コストリーダーシップ」「差別化」「集中（ニッチ）」の3つ。中途半端は最悪
- **5フォース分析**: 新規参入/代替品/買い手/売り手/既存競合 で業界の競争環境を評価

### 1-7. USP（Rosser Reeves, 1961）[C]
- **核心**: すべての広告は顧客に独自の提案をしなければならない。競合が言えない独自の主張を一つ見つけ、全コミュニケーションの軸にする
- **例**: M&M's「口でとろけて、手にとけない」

### 1-8. ブランドアーキタイプ（Mark & Pearson, 2001）[C]
- **核心**: ユングの12元型をブランドパーソナリティに適用。顧客との感情的つながりを構築
- **応用**: Nike＝英雄、Apple＝魔術師、Harley-Davidson＝反逆者。1-2個選びトーン・ビジュアル・ストーリーを統一

## SECTION 2: グロースマーケティング

### 2-1. グロースハッキング（Sean Ellis, 2017）[C]
- **核心**: 「このプロダクトが使えなくなったらどう感じるか？」で40%以上が「非常に残念」と回答すればPMF達成
- **応用**: PMF確認→成長レバー特定→高速実験サイクル→部門横断グロースチーム

### 2-2. コールドスタート問題（Andrew Chen, 2021）[C]
- **核心**: ネットワーク効果型プロダクトの5ステージ。最大の課題は最初のユーザー獲得
- **5ステージ**: Cold Start→Tipping Point→Escape Velocity→Hitting the Ceiling→Moat
- **事例**: Tinder＝大学パーティーからスタート、Facebook＝ハーバード限定

### 2-3. バイラル係数（K-factor）[B]
- **核心**: K = 招待数 × コンバージョン率。K>1で指数関数的成長
- **応用**: バイラルサイクルタイム（1サイクルの所要時間）の短縮も重要。Hotmailの署名が初期の代表例

### 2-4. プロダクト・レッド・グロース（PLG）[B]
- **核心**: プロダクト自体が顧客獲得・活性化・リテンションの主エンジン
- **応用**: Slack/Zoom/Notion/Figma。Time-to-Value（価値到達時間）の最小化が鍵
- **データ**: PLG企業はNon-PLG企業比でNet Dollar Retentionが高い（OpenView PLGインデックス）

### 2-5. Aha Moment [B]
- **核心**: ユーザーがプロダクトの真の価値を「体感する」瞬間。到達したユーザーのリテンション率は劇的に高い
- **事例**: Facebook＝10日以内に7人の友達追加、Slack＝2,000メッセージ送信、Twitter＝30人フォロー

### 2-6. ノーススターメトリック（NSM）[C]
- **核心**: 組織全体が追うべき単一の指標。顧客の「コア価値」を反映するメトリクス
- **事例**: Spotify＝再生時間、Airbnb＝予約泊数、Slack＝メッセージ送信数

### 2-7. パイレーツメトリクス AARRR（Dave McClure, 2007）[C]
- **核心**: Acquisition→Activation→Retention→Revenue→Referral の5段階。ボトルネック特定が鍵
- **応用**: 各段階の転換率をダッシュボード化し、最もインパクトの大きいところから改善

### 2-8. グロースループ vs ファネル（Reforge, 2018）[C]
- **核心**: ファネルは一方通行の加算モデル。ループは出力が入力になる循環モデルで複利的成長
- **3種のループ**: バイラルループ / コンテンツループ / ペイドループ（収益→広告→新規→収益…）

### 2-9. リテンションカーブとコホート分析 [A]
- **核心**: カーブが平坦化すればPMFの証拠。0に収束なら根本的問題
- **ベンチマーク**: SaaS Day30で40%以上、モバイルアプリ Day1で25%以上

### 2-10. LTV/CAC比率 [B]
- **核心**: LTV÷CACが3:1以上で健全。5:1以上は投資不足の可能性
- **応用**: CACの回収期間（Payback Period）は12ヶ月以内が目安

## SECTION 3: コンテンツマーケティング & SEO

### 3-1. They Ask, You Answer（Marcus Sheridan, 2017）[C]
- **核心**: 顧客が聞く質問にすべて正直に答えろ。競合が答えたがらない話題こそ書け
- **Big 5コンテンツ**: 価格・コスト / 問題点・欠点 / 比較 / レビュー / ベスト・オブ

### 3-2. ハブ&スポーク / トピッククラスター [B]
- **核心**: ハブ記事（包括テーマ）＋スポーク記事（個別トピック）を内部リンクで接続
- **応用**: Googleがテーマの専門性を認識→サイト全体のドメインオーソリティ上昇

### 3-3. E-E-A-T（Google品質評価ガイドライン）[A]
- **核心**: Experience（経験）/ Expertise（専門性）/ Authoritativeness（権威性）/ Trustworthiness（信頼性）
- **応用**: 著者プロフィール充実、実体験コンテンツ、権威サイトからの被リンク。YMYL分野で特に重要

### 3-4. スカイスクレイパーテクニック（Brian Dean）[D]
- **核心**: 被リンク多い既存コンテンツを見つけ→上位互換を作り→リンク元にアウトリーチ
- **Dean報告**: 検索トラフィック110%増

### 3-5. Gary Veeのピラー→マイクロコンテンツモデル [D]
- **核心**: 1つの長尺コンテンツから30-100個のマイクロコンテンツを派生させ全プラットフォームに展開
- **応用**: 1時間YouTube→5 Reels→20 Tweets→3ブログ→15 TikTok。AIツールで切り出し可能

### 3-6. 検索意図の4分類（Broder, 2002）[A]
- **核心**: Informational / Navigational / Transactional / Commercial Investigation
- **応用**: 意図に合ったコンテンツ形式でなければ上位表示されない

### 3-7. コンテンツフライホイール（HubSpot, 2018）[C]
- **核心**: コンテンツが集客→エンゲージメント→コンバージョン→顧客満足→口コミ→さらなる集客…の自己強化ループ
- **応用**: フライホイールの「摩擦」を取り除くことに注力

## SECTION 4: ダイレクトレスポンスマーケティングの古典

### 4-1. Scientific Advertising（Claude Hopkins, 1923）[C]
- **核心**: 広告は科学。テスト・計測・結果で判断。広告は「紙上のセールスマン」
- **応用**: ヘッドライン重視（本文を読むかは見出しで決まる）、サンプリング、A/Bテストの原型

### 4-2. David Ogilvyの原則 [C]
- **核心**: 広告の目的は売ること。リサーチに基づくクリエイティブ
- **応用**:
  - ヘッドラインに80%の予算価値（本文を読む人の5倍がヘッドラインを読む）
  - 長文コピーは関心高い読者向けに有効
  - 具体的事実: 「時速60マイルのロールスロイスで最も大きな音は電子時計」

### 4-3. AIDAモデル（Lewis, 1898頃）[B]
- **核心**: Attention→Interest→Desire→Action の4段階
- **注意**: 「必ずこの順序を経る」は現代研究では否定的だが、構成テンプレートとして有用

### 4-4. PAS（Problem-Agitate-Solution）[D]
- **核心**: 問題提起→苦痛を煽る→解決策を提示
- **変形**: PASTOR（Problem→Amplify→Story→Testimony→Offer→Response）

### 4-5. Dan Kennedy「No BS」の原則 [D]
- **核心**: すべてのマーケティングはダイレクトレスポンス。計測できないマーケティングはやるな
- **10ルール**: 必ずオファー / 今すぐの理由 / 明確な指示 / トラッキング / ブランディングにコストかけるな / フォローアップ / 強力なコピー / メールオーダー広告のように / 結果がすべて / 規律

### 4-6. Gary Halbertのレッスン [D]
- **核心**: 「飢えた群衆を見つけろ」— コピーの上手さよりリスト（ターゲット）が大事
- **応用**: A-pile（私信風）に入るDMを書け / ストーリーで80%構成 / P.S.は本文の次に読まれる

### 4-7. Jay Abraham「Strategy of Preeminence」[C]
- **核心**: 売り手ではなく「信頼されるアドバイザー」になれ。お金が動く前から価値提供
- **3つの成長戦略**: クライアント数を増やす / 取引単価を上げる / 購買頻度を増やす
- **リスクリバーサル**: 顧客のリスクをすべて売り手が負う。保証を強くするほどCVR上がり、返金率は上がらない

### 4-8. Eugene Schwartz — 認知レベル以外のフレームワーク

#### マーケット・ソフィスティケーション（市場成熟度）の5段階 [C]
1. **処女市場**: シンプルな約束だけで売れる
2. **メカニズム導入**: 「なぜ効くのか」説明が必要
3. **新メカニズム**: 既存と違う新手法で差別化
4. **ワンアップマンシップ**: 精緻化・誇張の競争
5. **市場アイデンティティ**: メカニズムでは差別化不能。アイデンティティ・感情・夢に訴求
- **せどり市場は第4-5段階**: ツールスペック比較では差別化困難。「さっさと一緒に稼ぐ仲間」的アイデンティティ訴求が有効

#### マス・デザイア [C]
- 広告は欲求を「作る」のではなく、既存の大衆の欲求を「チャネリング」して自社製品に向ける

#### 強化と識別 [C]
- 強化: 約束やメカニズムをより具体的・強力に（数値、タイムフレーム、保証）
- 識別: 読者が「これは自分のことだ」と感じる（日常の具体的描写、ペルソナの言葉遣い模倣）

### 4-9. StoryBrand / SB7（Donald Miller, 2017）[C]
- **核心**: 顧客をストーリーの「主人公」、企業を「ガイド」として位置づける
- **SB7**: 主人公→問題→ガイドに出会う→計画→行動を促す→失敗回避→成功

### 4-10. FABフレームワーク [C]
- Feature（機能）→Advantage（利点）→Benefit（利益）の順に翻訳
- 「リアルタイム価格比較機能(F)→巡回の手間省ける(A)→1日30分を他の仕入れに使える(B)」

### 4-11. 4U's ヘッドライン（Michael Masterson）[D]
- Useful（役立つ）/ Urgent（緊急性）/ Unique（独自性）/ Ultra-specific（超具体的）

## SECTION 5: オファーデザイン & マネタイゼーション

### 5-1. グランドスラム・オファー（Hormozi, 2021）[D]
- **核心**: 「買わないほうがバカ」レベルのオファー設計
- **価値方程式**: 価値＝（夢の結果 × 達成可能性）÷（時間 × 労力・犠牲）
- **構築ステップ**: ドリームアウトカム特定→障害リスト→各障害の解決策→パッケージ化
- **価値の増幅装置**: 保証 / スカーシティ / アージェンシー / ボーナス
- **価格設定**: 安売りするな。価値を10倍にして価格を上げろ

### 5-2. Russell Brunsonのバリューラダー [D]
- **核心**: 顧客を段階的に高価値・高価格の商品に導く階段構造
- **4段階**: リードマグネット/トリップワイヤー($0-7)→コアオファー($47-997)→プロフィットマキシマイザー($997-10K+)→リターンパス

### 5-3. トリップワイヤー戦略（Ryan Deiss）[B]
- **核心**: 低価格($1-20)で「最初の購入」の心理的障壁を越えさせる。一度買った顧客は2回目のハードルが劇的に下がる
- **根拠**: コミットメントと一貫性の原理（Cialdini）。トリップワイヤー自体は利益目的でなくLTVで回収

### 5-4. 収益モデルパターン

| モデル | 核心 | 証拠強度 |
|--------|------|---------|
| **フリーミアム** | 基本無料→2-5%が有料転換 | B |
| **サブスクリプション** | 月額/年額で予測可能な収益。チャーン率5%以下が健全 | A |
| **マーケットプレイス** | 買い手と売り手を仲介。テイクレート5-30% | A |
| **従量課金** | 使った分だけ。ベース+従量のハイブリッドが一般的 | B |

## SECTION 6: コミュニティ構築

### 6-1. トライブ理論（Seth Godin, 2008）[C]
- **核心**: 万人に売るな、情熱を共有する少数に集中しろ
- **応用**: 多方向コミュニケーション設計。「People Like Us Do Things Like This」が合言葉

### 6-2. 1,000 True Fans（Kevin Kelly, 2008）[C]
- **核心**: 年間$100払うファンが1,000人いれば年収$100,000
- **応用**: 広く浅くではなく深くパーソナルな関係構築。真のファンは最高のマーケティング部隊

### 6-3. コミュニティ主導型成長（CLG）[B]
- **核心**: ユーザー同士の交流・助け合いが価値を生み、リテンション・獲得が自然発生
- **事例**: Lego、Sephora、Peloton、Nike Run Club

### 6-4. ダークソーシャル [B]
- **核心**: コンテンツ共有の84%はDM・LINE・メールなど追跡不能なチャネルで発生（HubSpot）
- **応用**: メッセンジャー共有ボタン追加、UTMパラメータ付きリンク提供

### 6-5. アンバサダープログラム [C]
- **核心**: 熱狂的顧客をブランド大使に。双方向価値設計が鍵
- **KPI**: 1人あたりのコンテンツ生成量、リーチ、コンバージョン率

### 6-6. UGC（ユーザー生成コンテンツ）戦略 [B]
- **核心**: UGCはブランド発信より圧倒的に信頼される（Stackla: 信頼度2.4倍）
- **応用**: ハッシュタグ、フォトコンテスト、レビュー依頼→LP・広告・SNSに二次利用

### 6-7. Minimum Viable Audience（Seth Godin, 2018）[C]
- **核心**: 「できるだけ多くの人」ではなく「最小限の正しい人」に向けて作る

### 6-8. パープルカウ（Seth Godin, 2003）[C]
- **核心**: 注目に値するほど際立ったものでなければ、マーケティングしても意味がない
- **応用**: 商品・サービス自体に話題性を組み込む。マーケティング以前の問題

### 6-9. パーミッションマーケティング（Seth Godin, 1999）[C]
- **核心**: 顧客の「許可」を得てからマーケティングする。注意の希少性時代に不可欠

## SECTION 7: ローンチ戦略

### 7-1. プロダクトローンチフォーミュラ / PLF（Jeff Walker, 2005/2014）[D]
- **核心**: 「横向きのセールスレター」でローンチ前に段階的に価値提供し、当日爆発的売上
- **応用**:
  - PLC（Pre-Launch Content）: 3本の無料コンテンツで教育→信頼→証明
  - Open Cart / Close Cart: カート期間限定で緊急性
  - Launch Stacking: 各ローンチの実績が次に積み上がる複利効果
- **4種**: シードローンチ→内部ローンチ→JVローンチ→エバーグリーンローンチ

### 7-2. ウェイトリスト/期待値構築 [B-C]
- **核心**: 心理トリガー（独占性・FOMO・期待の快感）で需要を醸成
- **応用**: 紹介制度付きウェイトリスト（Dropbox方式）、バースト型コミュニケーション（3-5日集中が高CVR）
- **神経科学**: 欲しいものを「待つ」行為は受け取るのと同じ快楽中枢を活性化

### 7-3. ウェビナーファネル [D]
- **核心**: 90分無料セミナー形式で教育→最後に限定オファー
- **応用**: ライブ→リプレイ→エバーグリーン自動化の段階的展開

### 7-4. 5つの教育フレームワーク（日本ローンチ実務）[D]
- 目的の教育→信頼の教育→問題の教育→手段の教育→投資の教育
- ステップメール/LINE配信で5段階を体系的に実施

## SECTION 8: メール & シーケンスマーケティング

### 8-1. ウェルカムシーケンス [B]
- **核心**: 登録直後のメールが最もエンゲージメント高い
- **構成**: 歓迎+特典→ブランドストーリー→価値ハイライト→ソーシャルプルーフ→限定オファー

### 8-2. ソープオペラシーケンス（Brunson / Chaperon）[D]
- **核心**: 7-10通でドラマのように物語展開。各メールが次へのフック

### 8-4. カート放棄シーケンス [B]

- **第2通（12-24時間後）**: ソーシャルプルーフ追加

## SECTION 14: 説得の心理学（上級）

### 14-1. 精緻化見込みモデル / ELM（Petty & Cacioppo, 1986）[A]
- **核心**: 説得の「中心ルート」（論理・事実）と「周辺ルート」（感情・手がかり）
- **応用**: 関与度高い商品→データ・比較。低い商品→デザイン・雰囲気・有名人

### 14-2. 物語輸送理論（Green & Brock, 2000）[A]
- **核心**: ストーリーに「没入」すると批判的思考が低下し、メッセージを受け入れやすくなる
- **応用**: 広告をストーリー形式にして反論を減らす。主人公への感情移入が態度変容を増幅

### 14-3. 自己決定理論 / SDT（Deci & Ryan, 1985）[A]
- **核心**: 動機づけは「自律性」「有能感」「関係性」の3欲求で決まる
- **応用**: 選択肢を与え（自律性）、成功体験を提供し（有能感）、コミュニティに所属させる（関係性）
- **内発 vs 外発**: 内発的動機の方が持続的。ロイヤリティプログラムはポイントだけでなくステータス・体験も

### 14-4. 制御焦点理論（Higgins, 1997）[A]
- **促進焦点**: 「こんな未来が手に入る」（達成・成長・理想の自分に反応）
- **防止焦点**: 「これを逃すと損」（安全・安心・失敗しないに反応）
- **制御適合**: メッセージが受け手の焦点と一致すると説得力倍増

### 14-5. 解釈レベル理論（Trope & Liberman, 2003）[A]
- **核心**: 心理的距離が遠いほど抽象的に、近いほど具体的に思考する
- **応用**: LP上部で「Why」を語り、下部で「How」と具体的手順を示す

### 14-6. 時間割引（Ainslie, 1975）[A]
- **核心**: 将来の大きな報酬より目の前の小さな報酬を過大評価する（双曲割引）
- **応用**: 「3ヶ月後に10万円得する」より「今日500円お得」が効く。即時特典+将来ベネフィットのハイブリッド

### 14-7. 決定疲労（Baumeister, 1998）[A]
- **核心**: 意思決定を繰り返すと判断の質が低下し、デフォルト選択に流れる
- **応用**: 商品ラインナップ絞り込み、デフォルト選択の戦略的設定、購入プロセスの簡素化
- **注意**: エゴ消耗モデルは近年再現性議論あり

### 14-8. 感情ヒューリスティック（Slovic, 2002）[A]
- **核心**: 「気分がいい＝リスクが低い」と判断する。無関係な感情でも判断に影響
- **応用**: ポジティブ感情を喚起してから商品提示。明るい色調、笑顔の画像、楽しい雰囲気

### 14-9. 利用可能性ヒューリスティック（Tversky & Kahneman, 1973）[A]
- **核心**: 思い出しやすい事例ほど頻度が高い・重要だと判断
- **鮮明性効果**: 「売上50%向上」より「月収20万だった主婦が100万に」というストーリーの方が強い

### 14-10. スノッブ効果 & ヴェブレン効果 [B]
- **スノッブ**: みんなが持っているものは欲しくなくなる→限定版・プレミアムライン
- **ヴェブレン**: 高価格であること自体が価値（顕示的消費）→価格を下げないことがブランド維持

## SECTION 17: ニューロマーケティング

### 17-1. 色彩心理学 [A-B]
- **核心**: 「赤は情熱」的な単純対応表は科学的に弱い。効果は文脈・文化・個人経験に依存
- **重要**: 色そのものより「ブランドとの適合性」が重要（Bottomley & Doyle, 2006）
- **データ**: 色はブランド認知を最大80%向上させる可能性。脳は文字の意味処理の200ms前に色を処理

### 17-2. 音・音楽 [A-B]
- **BGMと滞留時間**: ソフトなBGMで最大30%延長
- **テンポ**: スローテンポ→移動速度低下→消費額増加（Milliman, 1982）
- **不一致効果**: スローテンポ+シトラス香り→消費40%減少。感覚の一貫性が重要
- **ソニックブランディング**: Intel、Netflix等。ブランド固有の音声アイデンティティ

### 17-3. 香りマーケティング [A]
- **データ**: 環境香りで平均消費額3-23%増加（671効果量のメタ分析）
- **事例**: フローラル香りでNikeの購買意欲84%向上
- **プルースト効果**: 嗅覚は大脳辺縁系に直結、他の感覚より強力に記憶と感情を喚起

### 17-4. アイトラッキング研究 [A]
- **F字型パターン**: 左上→右→少し下がって左→右→左端を縦にスキャン（Nielsen, 2006）
- **顔の視線誘導**: 人物の視線の先に注目が集まる→モデルの視線をCTAに向ける
- **バナーブラインドネス**: 広告に見える要素を無意識に無視→ネイティブ広告的デザイン
- **中央・左上配置の優位性**: CTAは中央配置、重要情報は左上

### 17-5. fMRI研究 [A]
- **感情的意思決定の優位性**: 購買決定の多くは感情プロセスが先行（Damasio, 1994）
- **報酬系と価格**: 好きなブランドの価格→報酬系活性化、嫌いなブランド→苦痛反応（Knutson, 2007）
- **応用**: ブランド好感度を先に構築→価格提示

### 17-6. ドーパミンと変動報酬 [A]
- **核心**: ドーパミンは報酬そのものではなく「報酬の予期」で最も放出（Schultz, 1997）
- **変動報酬**: 予測不能な報酬→固定報酬の2倍のドーパミン
- **応用**: ランダムクーポン、サプライズ特典、ガチャ的要素
- **ストリーク効果**: Duolingo 7日ストリーク達成者は長期継続率3.6倍

### 17-7. ミラーニューロンと共感 [A-B]
- **核心**: 他者の行動を観察するだけで、自分が同じ行動をしているように脳が活性化（Rizzolatti, 1996）
- **応用**: 商品使用シーンを「追体験できる」映像化。「モデル」より「リアルな人」の使用シーン

- **新市場型**: 非消費者を新たに顧客化する

### 18-3. フライホイール効果（Jim Collins, 2001）[C]
- **核心**: 好循環が回り続けることで加速度的に成長
- **Amazonモデル**: 低価格→顧客体験向上→トラフィック増→出品者増→品揃え拡大→さらなる低価格
- **応用**: 自社の好循環要素を特定し、最も回しやすい一点に集中投資

### 18-4. リーンスタートアップのマーケティング応用 [C]
- **MVP**: LP1枚+広告で需要検証→反応があれば本格展開
- **Build-Measure-Learn**: 仮説→実験→計測→学習の高速サイクル
- **バニティメトリクスの罠**: フォロワー数やPVではなくCVR・LTV・CACを追え

### 18-5. パレートの法則（80/20）[A]
- **核心**: 売上の80%は20%の顧客から、利益の80%は20%の商品から
- **応用**: トップ20%に最もリソース配分。80/20の中の80/20（64/4の法則）でさらに絞る

## SECTION 19: 実世界の成功パターン & アンチパターン

### 19-1. ソロクリエイターの成功パターン [D]
- **レバレッジ型への移行**: 時間切り売り→デジタル商品・講座・メンバーシップが分水嶺
- **ニッチ×専門性の集中**: 「何でも屋」を捨て、一つの明確な専門領域に絞る
- **収益源の分散**: 3つ以上の収益チャネル
- **「ノー」と言う能力**: 核となる強み以外は断るか外注

### 19-2. 収益マイルストーンと各段階の変化 [D]
| 段階 | 特徴 | やること |
|------|------|---------|
| 0→月10万 | コンセプト検証期 | 少人数への直接販売。DM/対面で1対1クロージング |
| 月10万→50万 | 仕組み化の始まり | LP+ステップメール+自動ウェビナー構築 |
| 月50万→100万 | 商品ラインナップ拡充 | フロント→バックエンド設計、アップセル導入 |
| 月100万→300万 | チーム化の判断 | 自分の時給以下の仕事は全て委任 |
| 月300万以上 | システム化と組織化 | SOP整備、権限委譲。「個人名」→「事業名」 |

### 19-3. Dan Koeの3段階マネタイズ [D]
- Stage1: サービス販売（0-5K/月）
- Stage2: デジタル商品（5-20K/月）
- Stage3: ブランド＋自動化（20K+/月）

### 19-4. オンラインビジネスのアンチパターン [D]
- **完璧主義の罠**: 70%の完成度で出してフィードバックベースで改善
- **プラットフォーム依存**: SNSは入口、リストが資産。必ず自社リスト持て
- **バニティメトリクス固執**: フォロワー10万でも収益ゼロは珍しくない
- **価格の底辺競争**: 安さではなく「結果」で差別化
- **コンテンツ燃え尽き**: バッチ制作、リサイクル、エバーグリーン比率を増やす
- **一人ですべてやる**: 強み以外は自動化or外注

### 19-5. コミュニティ収益化パターン [D]
- **ラダー**: 無料参加→低額コンテンツ→有料メンバーシップ→高額サービス
- **「早すぎ」問題**: 信頼構築前のマネタイズはコミュニティ崩壊を招く。最低3-6ヶ月は無料で価値提供
- **コース×コミュニティ併用**: 構造化された学習＋非構造化された交流の組み合わせが最強
