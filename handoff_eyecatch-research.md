## リサーチ結果（2026-03-25）

### アイキャッチ改善の効果

さっさのケース（SNS流入が起点）ではアイキャッチ改善の優先度は高い。

| 条件 | 効果 | エビデンス強度 |
|------|------|---------------|
| OGP画像あり vs なし | エンゲージメント2〜3倍 | 中（BuzzSumo 2015-2018、Twitter公式 2014） |
| OGP画像の質を改善 | SNS経由CTR +20〜40% | 弱（マーケター経験値集約。RCTなし） |
| 顔あり vs なし | CTR +20〜30% | 中（Taboola/Outbrain内部分析。広告PFなので利益相反あり） |
| Google Discover | 大画像でCTR +5〜10%pt | 中（Google公式） |
| YouTubeサムネ改善 | CTR +20〜40%（相対値） | 中（TubeBuddy/vidIQ） |

エビデンスの質は全体的に低い。学術RCTはほぼなく、マーケツール会社や広告PFの自社分析が中心。ただしSNS流入が起点なら、やらない理由もない。

### X（Twitter）OGP表示仕様（重要）

**2024年10月以降の仕様変更: OGPからタイトルテキストが消えた。画像のみ表示される。**
→ **画像自体にメッセージを入れないと、何の記事か伝わらない。**

- Summary Large Image: 1200x630px、中央基準でクロップ、角丸
- 安全領域: 上下左右から約10%内側
- スマホ表示: 約506x265px。この状態で文字が読めるか確認必須
- ダークモードユーザーが多い → 純白背景は避ける
- リンク付き投稿はリーチが絞られる傾向 → OGP画像の質がCTRに直結

参考: ベーコンさんの世界ブログ「XのOGPデザイン3つの方法」

### 有料コンテンツのサムネイル傾向（note/Brain）

せどり系発信者のデザインは参考にならない（まともにやってる人がいない）。ジャンル問わず有料コンテンツで売れてるサムネの傾向：

| 要素 | 傾向 |
|------|------|
| 配色 | **黒背景×金・白テキスト**が高単価・秘匿性演出の定番。赤×黒も多い |
| テキスト | 必ず入れる。数字+秘匿性が鉄板（「月収○万」「○年間隠していた」「○部突破」） |
| 構図 | テキスト主体。ノウハウ系は写真よりテキスト重視が多い |
| フォント | 太ゴシック体。小さい文字は使わない |
| 文字数 | **13文字以内が理想**（一瞬で理解できる上限） |
| 枠（ボーダー） | サムネに縁をつけるとタイムラインで画像の境界が明確になり目立つ |

#### ノウハウ系サムネの原則（桜川和樹の考察）
- ノウハウ系は「イメージによる補完が難しい」のでテキスト主体にならざるを得ない
- 効果的なサムネに必要なのは**「具体性」と「期待感」**
- 何が書かれているかが一目でわかる + 自分に関係ありそうと思える

### YouTubeサムネ 2025-2026トレンド

- **3語以内の超短文**（情報詰め込み→極限まで減らす方向にシフト）
- 顔芸+感情表現（顔がサムネの40-60%）
- ビフォーアフター/対比構図
- **赤・黄・黒の3色支配**（パステル調はCTR低い）
- **AI非感（手作り感）**: AI画像が増えた反動で、あえて雑に見えるデザインがCTR高い
