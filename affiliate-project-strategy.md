- **スタンス**: 店舗せどりはプロ / 電脳は初心者

```

3000円→35000円
```

**Step2: リプで誘導**
```

仕入れの解説とか、質問はこっちで↓
[オプチャURL]
```

**Step4: ローンチ**
- 自然な流れでツール紹介

### 刷り込みたいメッセージ：「店舗→電脳は地続き」

**核心**
- 店舗せどりのノウハウは電脳でもそのまま流用できる
- 「商品自体の価値を読む」スキルは店舗も電脳も共通
- 知識のコスパがいい（1から学び直す必要がない）

**店舗せどりの2つの方向性**
1. 店舗の癖を読んで利益商品を探す（店舗特有）
2. 商品自体の価値を読んで仕入れる（店舗・電脳共通）←これが活きる

**差別化ポイント**
- 新品せどりは型番・相場勝負。商品知識の積み上げがしにくい
- 中古せどりは目利き力が資産になる。だから「地続き」が成立する

**刷り込み例**
```
店舗で「このぬいぐるみ、タグ付きで状態いいから高く売れる」って判断してるでしょ？

電脳でも同じ。
セカストオンラインで見つけた時も、同じ目線で判断できる。

店舗のノウハウ、そのまま使えるから電脳のハードル思ったより低いよ
```

## Xアナリティクス分析結果（2025/06〜2026/02）

### バズの法則（インプ100万超えポストの共通点）

| インプ | 内容 | 要因 |
|--------|------|------|
| 930万 | シュガーバニーズ 1000円→5〜20万円 | 「実家に眠ってるかも」訴求 |
| 790万 | ぶちくま引用 1320円→42万円 | 価格差が異常 |
| 410万 | ちいかわ抽選 | 話題性 |
| 190万 | 缶バッジ 407円→13万円 | 「即売れ」強調 |
| 170万 | 汚れアンコレ16万円 | 「汚れてるのに」意外性 |
| 130万 | 缶バッジ 660円×2→28万円 | 価格差+シンプルデザイン |
| 100万 | F賞4000円 | 「ハズレなのに」意外性 |

**バズ要素：**
- 「〇〇円→〇〇万円」のフォーマット
- 「〜なのに」の意外性（F賞なのに、汚れてるのに）
- 価格差10倍以上
