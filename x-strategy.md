Xの投稿設計・分析・改善のための戦略フレームワーク。データ分析に基づく暗黙知を体系化。

## 2. バズ公式

```
バズ確率 = IP認知度 × 自分事化度
```

### IP認知度の階層

| 階層 | IP例 | ROM専母数 | 単純コピペ |
|------|------|----------|-----------|
| **S（国民的）** | ポケモン・サンリオ・ジブリ・鬼滅・ドラえもん | 数千万 | 自分事化あれば伸びる |
| **A（世代的）** | ハンター・ハイキュー・プリキュア・どうぶつの森・リラックマ | 数百万 | 工夫次第 |
| **B（層限定）** | ダンガンロンパ・ペルソナ・銀魂・忍たま | 数十万 | 画像・切り口の工夫必須 |
| **C（コア）** | 特撮・イナイレ・第五人格 | 数万 | 単独では厳しい |

### 自分事化フレーズ集

| フレーズ | 効果 | 実績例 |
|---------|------|--------|
| 「これ家にあったら終わりです」 | 危機感+期待 | 930万imp（シュガバニ） |
| 「実家の押し入れ、もう一度だけ見て」 | 行動喚起 | 930万imp（シュガバニ） |
| 「まだ持ってたらガチで勝ち組」 | 所有確認欲求 | 930万imp（シュガバニ） |
| 「これ、捨ててなかったら勝ち」 | 後悔の感情 | 28,000imp（どう森） |
| 「当時◯◯で普通に売ってた」 | 購入経験の想起 | 930万imp（シュガバニ） |
| 「定価◯円で買えたやつ」 | 当時の記憶呼び起こし | - |
| 「UFOキャッチャーで取ったな」 | 体験記憶の想起 | 17,000imp（ソニック） |

### 100万インプ超えの共通パターン（実データ）

| インプ | 内容 | キーフレーズ | 推移型 |
|--------|------|-------------|--------|
| 930万 | シュガバニ | 「これ家にあったら終わりです」 | 遅延型 |
| 790万 | ぶちくま引用 | 「即売れえっぐい 1320円→42万円」 | - |
| 410万 | ちいかわ抽選 | 「おいおい、嘘だろ…」 | 初動特大 |
| 190万 | 銀魂缶バッジ | 「即売れえっぐい 407円→13万円」 | 初動ピーク |
| 170万 | 汚れモンコレ | 「ここまで汚れてるのに16万円」 | 超初速 |
| 130万 | 缶バッジ660円×2 | 「特徴ないデザインなのにここまで高額」 | 二峰 |
| 100万 | F賞一番くじ | 「F賞なのに4000円で売れててエグい」 | 初速 |

**メガバズの法則**:
- 「〜なのに」の意外性が最強（汚れてるのに、F賞なのに、特徴ないのに）
- 価格差フォーマット「◯◯円→◯◯万円」は鉄板
- 「即売れ」は需要の証明として機能
- 遅延型は画像精査を誘発する設計が必須

## 3. 二軸戦略

### 投稿タイプ

| タイプ | 目的 | ターゲット | 期待インプ | 頻度 |
|--------|------|-----------|-----------|------|
| **大バズ狙い** | 認知拡大・フォロワー獲得 | 一般層 | 10万〜数百万 | 週1〜2 |
| **中間安定型** | ファンベース維持・信頼構築 | せどり・物販層 | 1〜3万（安定） | 週2〜3 |

### 大バズ狙いの設計

- **SランクIP × 自分事化フレーズ**
- 価格差のインパクト（定価→現在価格、100倍以上が理想）
- 画像を「精査させる」設計（複数商品・価格差一覧・レア箇所ハイライト）

### 中間安定型（まとめ系）のテンプレート

| フォーマット | 例 |
|-------------|-----|
| 【これだけ見て】◯◯ | 原画デザイン缶バッジ |
| 【見つけたら勝ち確】◯◯ | パンスト、ダイヤのA |
| 【保存版】◯◯選 | プライズ50選 |
| ◯◯まとめました | 高額缶バッジ |
| 【リサイクルショップ攻略】◯◯ | キティぬいぐるみ |

**中間安定型の価値**:
- 高ブックマーク率 → Grokの「質」評価に貢献
- アカウント全体の信頼スコア向上
- 大バズが来たときの受け皿（フォローされる理由）

## 4. リライト戦略

### 基本原則

旧アルゴ期（〜2025/10）で「相対的に高かった」投稿は、新アルゴ（2025/11〜）でリライトする価値が高い。

### 成功/失敗の分岐点

| 結果 | パターン | 理由 |
|------|---------|------|
| **大成功** | 画像・切り口を変更 | 「新しい情報」として再評価 |
| **大成功** | SランクIP × 自分事化追加 | IP認知度で押し切れる |
| 微増 | 金額更新で鮮度維持 | 最新履歴の付加価値 |
| 横ばい | 内容変更なし | リライトの意味薄い |
| **減少** | 同内容の単純再投稿 | Grokが「既視感」を検出 |

### リライト時のチェックリスト

- [ ] 画像を変更したか？（単品→複数比較、全体俯瞰など）
- [ ] 金額を最新に更新したか？
- [ ] 自分事化フレーズを追加したか？
- [ ] 切り口を変えたか？（「エグい」→「実家に眠ってるかも」など）

## 5. 推移型と狙い方

### 推移型別パフォーマンス

| 推移型 | 平均インプ | 特徴 | 狙い方 |
|--------|-----------|------|--------|
| **遅延型** | 最高 | 18h以降に伸びる | 画像精査誘発・SランクIP |
| 超初速型 | 高 | 15分以内が命 | 話題性・速報系 |
| 二峰型 | 中〜高 | 初速+再拡散 | 安定狙い |
| 初速型 | 低〜中 | 従来型 | 旧アルゴ時代の主流 |

### 遅延型を狙う設計

- 画像を「精査させる」（複数商品・細部に情報）
- 滞在時間を稼ぐコンテンツ
- 引用RT戦略（他者投稿に便乗→遅延型狙い）

## 6. 引用RT戦略（便乗型）

### 成功パターン

| インプ | 元ネタ | 追加した価値 |
|--------|--------|-------------|
| 790万 | シュガバニ投稿 | ぶちくまの写真追加「1320円→42万円」 |
| 67,000 | 忍たまほわぬい | 「ブックオフで買ったことあります！」+写真 |
| 54,000 | amiibo紹介 | 「ハードオフのamiibo安いですね 先月みっけました」 |
| 48,000 | たまごっち紹介 | 「爆速で売れちゃってるからもっと高く売れそう」 |

### 引用RTの鉄則

1. **情報を追加する**（単なる感想はNG）
2. **自分の経験/写真を添える**（「買ったことある」「先月見つけた」）
3. **価格情報を補足する**（「もっと高く売れそう」「◯◯円で販売可能」）
4. **訂正・補足で価値を出す**（「140万で売れたポムポムプリンなんて無いじゃん」→33,000imp）

### 引用RTのNG

- 「すごい！」だけの感想
- 元ネタと同じ情報の繰り返し
- 自分語り（「俺もやってみよう」系）

## 7. 避けるべきパターン

| NG | 理由 | 改善案 | 実例 |
|----|------|--------|------|
| 外部リンク直貼り | 致命的ペナルティ | X内完結コンテンツに | - |
| 問いかけ形式（「〜じゃない？」） | 断言の方が伸びる | 断定形に変更 | ポムポムプリン16k（問いかけ）vs類似断定30k+ |
| 特撮単独投稿 | ROM専母数が少ない | 他ジャンルと複合 or 引用RT | 特撮単体7.8k vs 他ジャンル30k+ |
| スクショ1枚 | 情報密度低い | 複数商品比較画像 | - |
| 限定◯名系（自分事化なし） | 共感されにくい | 「当時売ってた」系に変換 | - |
| 自己言及メイン | 自慢に見える | 商品フォーカスに | 成約ポスト3k imp（これは例外） |

### 価格帯の閾値

| 価格帯 | 効果 | 備考 |
|--------|------|------|
| **10万円以上** | 高インパクト | 単品で成立 |
| **価格差100倍以上** | 最強 | 「407円→13万円」（190万imp） |
| **価格差10倍以上** | 十分 | 最低ライン |
| 44万円（ハイキュー） | やや弱い | スクショの見せ方で伸び悩み |
| 40万円未満 | インパクト不足 | まとめ系にするか切り口工夫 |

### 成約とバズは別物

| 目的 | インプ目安 | 内容例 |
|------|-----------|--------|
| **バズ（認知）** | 10万〜数百万 | 「407円→13万円！！」 |
| **成約** | 3,000でも可 | 「月5万安定の教科書」→3件成約 |

**重要**: インプが低くても成約につながるポストがある。役割が違う。

## 8. 分析時のチェック観点

CSVデータを分析する際は以下を確認：

1. **時期別パフォーマンス**（アルゴリズム変更の影響）
2. **推移型別の傾向**
3. **IP認知度と自分事化度の相関**
4. **ブックマーク率**（0.3%以上で優良）
5. **リライト候補**（旧アルゴ期の相対的高パフォーマンス）
6. **中間安定型の貢献度**

## 10. さっさの文体特徴

### ターゲット別文体

| ターゲット | 文体 | 特徴 |
|-----------|------|------|
| **外部バズ（一般向け）** | 敬語なし・短い・感嘆のみ | 「エグい」「高すぎ！！」「マジで穴場」 |
| **内輪（せどらー向け）** | 敬語あり・説明調 | 「〜ですね！」「〜かなと」「覚えておきたい」 |

### 頻出フレーズ

| 要素 | 使用例 |
|-----|--------|
| 感嘆 | 「エグい」「エッグい」「やばすぎ」「高すぎる」 |
| 語尾 | 「〜笑」「〜ですね！」「！！」多用 |
| 意外性 | 「〜なのに」「マジで意味わからん」 |
| 行動促し | 「ワンチャン！」「覚えておきたい」 |
| ヘッドライン | 【】で括る |
| 価格差 | 「○○円→○○円！！」 |

### 使わない表現（NG）

- 「知ってた？」系 → 商材臭い
- 命令形「〜してください」
- ビジネス臭い敬語
- 問いかけ形式「〜じゃない？」 → 断定の方が伸びる
- 自己言及（「電脳強化中」「頑張ってる」系）→ 自慢に見える

### プロセス共有の正しいやり方

**OK（商品フォーカス）**:
- 「ブックオフで買ったことあります！」+写真 → 67,000imp
- 「やっすい」（単純な感想）→ 7,800imp
- 「先月みっけました」+写真 → 54,000imp

**NG（自己フォーカス）**:
- 「電脳せどり強化中です」
- 「店舗と同じ目線でいける」
- 「本気で取り組んでます」

**違い**: OKパターンは「商品の話をしてたら自分が出てきた」。NGパターンは「自分の話をしてる」。

### テンプレート

**バズ狙い（外部向け）**
```
[サイト名]、穴場すぎる

[仕入れ値]円→[販売価格]円！！

[判断理由1文・敬語なし]
```

**フォロワー向け（内輪）**
```
[内容]、思ったより[結果]ですね

[詳細]
[学びや気づき・敬語あり]笑
```

**セルフリプ（誘導）**
```
[活動内容]、本気で強化中です

[提供価値]あればこっちで！
[URL]
```

## 11. なぜ短文なのか（理論的根拠）

### アルゴリズム観点

- **完読率が重要** → 短い方が最後まで読まれる
- 画像で滞在時間を稼ぐ設計と組み合わせる
- 長文は離脱率が上がり、エンゲージメント率が下がる

### 認知心理学観点

- **処理流暢性**: 読みやすい = 信頼・好感が上がる
- 1秒で理解できる → シェアのハードルが下がる
- 認知負荷が低いほど行動（いいね・RT）に繋がりやすい

### 実データ

| インプ | 文字数 | 内容 |
|--------|--------|------|
| 930万 | 3行+自分事化 | シュガバニ |
| 790万 | 1行+価格差 | ぶちくま引用 |
| 190万 | 2行 | 銀魂缶バッジ |

**長文説明系は伸びにくい**（データで確認済み）

### 結論

説明を足すと「親切」ではなく「邪魔」になる。

- 補足したくなったらセルフリプで分離
- 本文は感嘆+価格差のみで完結させる
