# X(Twitter) アナリティクス データ抽出スキル

Xのポストアナリティクス画面のスクリーンショットから、Googleスプレッドシートへ直接貼り付け可能な形式でデータを出力します。

---

## 1. 目的

- **完全取得**: 指定された全データ項目を欠損なく保持
- **表示用再構成**: Googleスプレッドシートへ直接貼り付け可能な形式で出力

---

## 2. 入力

1. **スクリーンショット**: Xのポストアナリティクス画面の画像
2. **文章**: ポストの本文（ユーザーが提供）
3. **補足**: ポストの補足説明（ユーザーが提供）

---

## 3. 必須取得カラム（COLUMN_ORDER）

以下の順序で **全項目を必ず取得・保持** してください。

```
日付
ポストURL

```

---

## 4. スクリーンショットからの読み取り対応表

| 画面上の表示 | 対応カラム |
|-------------|-----------|
| 投稿日時（例: Dec 31, 2025） | 日付 |
| 画面下部のURL または ステータスバーのURL | ポストURL |
| 棒グラフアイコン横の数値 / Views / Impressions | インプレッション |
| ハートアイコン横の数値 | いいね |
| 吹き出しアイコン横の数値 | リプ |
| リポストアイコン横の数値（矢印2本） | リポスト |
| 人型アイコン横の数値 / Profile clicks | プロフクリック |
| リンクアイコン横の数値 / Link clicks | クリック数 |
| New followers / フォロワー増減表示 | フォロワー増減 |
| Engagement rate / ％表示 | エンゲ率 |
| 再生アイコン横の数値 / Video views | 動画再生 |
| 「最初の48時間」グラフのパターン | 推移型 |
| 旗アイコン横の数値 / Bookmarks | ブックマーク |
| 共有アイコン横の数値 / Shares | シェア |

---

## 5. 欠損データの扱い（厳守）

**空欄は禁止** です。データが存在しない場合：

| データ種別 | 埋め方 |
|-----------|--------|
| 数値系（インプレッション、いいね等） | `0` |
| URL | `推定：取得不可（画面に表示なし）` |
| 不明情報 | `推定：◯◯（理由）` |

---

## 6. 数値変換ルール

| 入力形式 | 出力形式 |
|---------|---------|
| 41K | 41000 |
| 3.3K | 3300 |
| 1.2M | 1200000 |
| 8.3% | 8.3% |

小数点以下は四捨五入してください。

---

## 7. 日付フォーマット

| 入力例 | 出力 |
|--------|------|
| Dec 31, 2025 | 2025-12-31 |
| 2025年12月31日 | 2025-12-31 |
| 12/31/25 | 2025-12-31 |

---

## 8. 推移型の判定基準

「最初の48時間」グラフのパターンから判定：

| パターン | 推移型 |
|---------|--------|
| 最初の数時間で大きくスパイク → 急減 | 初速型 |
| 長時間にわたり安定 | 持続型 |
| 投稿から時間経過後にピーク | 遅延型 |
| 複数のピークが存在 | 二峰型 |
| 全体的に低インプレッション | 低調型 |

---

## 9. 表示用マッピング（出力時の列名）

| 内部項目 | 表示名 |
|---------|--------|
| リプ | リプライ |
| クリック数 | リンククリック |
| 文章 | 投稿内容（事実ベース1文） |
| 補足 | 補足（希少性・価格等の事実） |

---

## 10. 禁止事項（厳守）

以下は **一切行わない** こと：

- 判断・評価・考察の追加
- 列の追加・削除
- 空欄の生成
- 感情文・主観表現

---

## 11. 出力形式

### ヘッダー行（1行目）


### データ行（2行目以降）

---

## 12. 実行手順

1. スクリーンショットから全数値を読み取る
2. 欠損があれば規定値で埋める
3. 数値変換・日付変換を適用
4. 推移型を判定
5. ユーザー提供の「文章」「補足」を組み込む
6. タブ区切り形式でコードブロック内に出力

ユーザーはこの出力をそのままGoogleスプレッドシートに貼り付けて使用します。
