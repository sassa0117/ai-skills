## 1. 合宿概要

| 項目 | 内容 |
|------|------|
| 開催日 | 2026年4月5日（1日間集中） |
| テーマ | 女性陣強化（前回参加者のスキル・マインド強化） |
| 対象者 | 明日香 / れいちゃん / まっすー（確定）。チェルは費用面で未確定 |
| 方針 | リンクに寄せない。個別スキル強化 + アシスタント連携 |

### 合宿の核心
3名の共通課題「**思考過多・完璧主義・現状満足**」を、パワーワーク中心の瞬発型プログラムで打破する。思考を削り、初動スピードと回転数を体に染み込ませる。

### プログラム設計パラメータ

| パラメータ | 値 |
|----------|------|
| ワーク数 | 約10本 |
| 1ワーク時間 | 20分〜最大1時間 |
| ロープレ時間 | 通常の1/3に短縮（例: 3分→1分） |
| 延長 | なし（厳守） |
| 構成比率 | ディスカッション75% / 瞬発課題25% |

### ワーク設計の意図
- パワーワーク中心（**思考ゼロ前提**、瞬発力重視）
- 「やりたくないことをやらせる」ワーク多め
- 合宿のテンション・推進力で乗り切る経験を重視
- 思考はディスカッションまとめで発揮（それ以外では思考を使わせない）
- 内省時間を与えずワーク実施 → 後から振り返りで回収

## 2. アシスタント行動基準

### NG（絶対にやらないこと）

| # | ルール | 詳細 |
|---|--------|------|
| NG1 | 直接指摘の禁止 | 「スピード感を意識して」等のワードを使わない |
| NG2 | 原因分析の禁止 | 「スピード感がないから」という分析を伝えない |
| NG3 | 普段できることを褒めない | 前半は特に、既存スキルの承認を控える |
| NG4 | 後出し戦略の過度な承認 | 後半を選ぶ行動・負荷かけた行為を褒めない |

### OK（積極的にやること）

| # | ルール | 具体例 |
|---|--------|--------|
| OK1 | 初手を出す行動を承認 | 「まとめられるなら初手で出す方がレベル上がる」 |
| OK2 | 「できそうだけどね」承認 | 「それ本当にできなかった？できそうだけどね」 |
| OK3 | 行動後に守りを結びつける | 出ちゃった行動に合わせてさりげなく伝える |
| OK4 | 小さな変化を拾う | 携帯を見なくなった等、微細な成長を認める |

### 矛盾回避の原則
- 初動スピード重視と質向上戦略が矛盾しないようNG基準を守る
- 「まとめられるなら初手で出す」ことで全体レベルが上がる、という方向で承認
- 「出ちゃった行動」に合わせて後から守りの話を結びつける（意図的に見せない）

## 3. レクチャー設計（キーワード結びつけ方式）

### フロー
1. **ワーク実施**（思考ゼロで体験）
2. **キーワード提示**（例: チームワーク、リーダーシップ）
3. **参加者がまとめる**（自分の言葉で言語化）
4. **言い回し変更ルール**（前の人と同じ表現NG）

### 言い回しルール

| 種別 | 例文 | 理由 |
|------|------|------|
| **NG** | 「チームワークを**大事にすることで**〜」 | 汎用的・他人事・思考が浅い |
| **OK** | 「**私に足りないのは**チームワーク**と気づいた**」 | 自分事・内省・具体的 |

### 狙い
- ワーク体験と学びの**結びつけ能力**を強化
- 同じ内容でも表現を変えることで、言語化の瞬発力を鍛える
- 初手で手を挙げる行動を促進（後出しで質を上げようとする傾向を是正）

## 4. 個別プロファイル分析

### れいちゃん
- **タグ**: 所属欲求 / 現状満足 / スキル強化型

**課題（なぜスピードが出ない）**
- サラリーマン寄り思考でリンクを継続
- コミュニティ所属欲求が主な動機
- 「やろうと思えばできる」が、やろうと思ってない
- 短期回収意識が弱い

**アプローチ**
- **マインドではなくスキル増強**がメイン
- 紹介後の関係構築力（橋渡し役）を鍛える
- 馴染んでない子を人と仲良くさせる練習
- 小さな変化（携帯非使用・合いの手）を承認
- 数打つ力は尊敬に値する。そこを活かす

**変化の兆し**
- 携帯いじらなくなった
- 飲み会で合いの手するようになった
- 嫌われないよう意識するようになった（前回合宿の効果）

**矛盾点**: ワッシュ退会にショック受けつつ「仕方ない」と言う。月会費低下は気にしている

**キャラと内面のズレ**

## 5. スピード獲得フロー（設計意図）

```
STEP1: パワーワーク（思考ゼロ・瞬発）
  ↓
STEP2: キーワード結びつけ（言語化・内省）
  ↓
STEP3: ディスカッション（まとめ・承認）
  ↓
STEP4: 次ワークへ即移行（回転数UP）
  ↓
（STEP1に戻る × 約10本）
```

**狙い**: 未完成でも初動を出す → 試す → 回収するサイクルを1日で体に叩き込む。長期前提の先延ばしを是正し、「半年考えて良いものを作る」ではなく「今日出して今日回収する」体験を積ませる。

### 連携の原則
- だいが感情で押す → りゅうが論理で受ける、の補完構造
- ディスカッションのまとめ言葉に思考力を集中（そこでしか使えない）
- ワーク自体は事前知識なしで参加し、レクチャー時間で思考負荷をかける

## 13. ワーク詳細（確定版）

### ① キャッチ（アイスブレイク）
| 項目 | 内容 |
|------|------|
| 所要時間 | 5〜10分 |
| ルール | 円になり、左手で輪っか（OKサイン）、右手の人差し指を隣の輪に入れる。「キャッチ」の合図で左手は捕まえ、右手は抜く。同時進行 |
| レクチャー | **なし**（純粋なアイスブレイク） |
| アシスタント | 全員参加。女性同士だからできる距離感のワーク |
| 意図 | 脳の右左同時使用。導入のテンション作り |

### ② 癖強自己紹介 + 自己主張度チェックシート
| 項目 | 内容 |
|------|------|
| 所要時間 | 20〜30分 |
| ルール | 山手線テンポで回す。一度出たフレーズは使用不可。アシスタント含め約20周（計100個） |
| 進行 | **STEP1**: 普通の自己紹介（「うさぎが好きです」等）を1周 → **STEP2**: 癖強自己紹介を回す → **STEP3**: 他人の癖強をどれだけ覚えているか紙に記入（ルール説明では明かさない） |
| 統計的狙い | 癖強の方が記憶に残る → 記憶に残る自己紹介の価値を体感 |
| 自己主張度チェックシート | **事前**（日常の自分）と**事後**（今日のワーク中の自分）の2回記入。差分で「自分の認識と実際の成果は違う」ことを可視化 |
| レクチャー方向性 | 発信寄り。「聞く方も大事だよ」にさらっと触れる程度 |
| 役割 | **主張担当: だい / 聞く担当: りゅう** |

**ワークの深層意図**:
- 自分の好きなことを言っても人は聞いていない → 覚えてもらうには工夫が必要
- 「目的や自分の思い ＝ 成功ではない」に繋がる
- 無意識下で何が出るか → 才能の開花
- ビジネスにおいて自己主張度は**チャックのように開け閉め**できるべき

### ③ 聖徳太子ゲーム
| 項目 | 内容 |
|------|------|
| 所要時間 | 40分（最大1時間） |
| ルール | お題（例: 果物の名前）に合わせ全員同時発言 → リーダーが誰が何を言ったか当てる |
| 難易度 | 3段階。単語（デモ）→ 難しい単語 → 文章（「最近あった嫌なこと」等） |
| 核心 | **ゾーンに入る体感**。簡単にクリアさせない。繰り返すうちに「聞こえるようになってきた」実感を持たせる |
| アシスタント | 意図的に難易度を上げる（小さい声で話す等）。参加者側の工夫（ボリューム調整、高音/低音の使い分け）を促す |
| 役割 | **だい: 承認ベース**（ポジティブな発言・工夫をした人を承認） / **りゅう: 日常置き換え**（他メンバーのリアクションを褒める、聞く姿勢の指摘） |

**承認の連携パターン**:
1. れいちゃんが「みんなのボリューム合わせよう」と発言 → **だいが承認**
2. あすかがそれに即座に合わせてボリューム調整 → **りゅうが「合宿でできてるけど日常でやってないよね」と優しく刺す**
3. このパターンで承認→日常への結びつけを自然に行う

**りゅうへのお願い（だいより）**:
- 主張が強い流れになって「聞く」にフォーカスされてない場合、「どういう声が聞きやすいか」という客観的な振りをしてほしい
- 承認ベースで動くだいには言いづらい部分を、りゅうが担う

### ④ 写真パズル
| 項目 | 内容 |
|------|------|
| 所要時間 | 30〜40分 |
| ルール | チラシや絵を切ったピースを各人に配布。**口頭のみ**でコミュニケーションし完成させる（並べるのは最後） |
| アシスタント | 参加しない（3人でやる） |
| NG | 雰囲気が悪くなること |
| 素材 | アニメポスター等（参加者がアニメ好きなため） |

**前回比較の活用**:
- 前回の「地図ワーク」でれいちゃんがもっと前のめりにリーダーシップ取れたはず → 今回は前回参加者のみだからその比較が可能
- れいちゃんが覚えていて行動した場合 → 承認
- 覚えていなかった場合 → 「前回で前回だけの人って言ったんだから」と言える

**感情設計（だいメモより）**:

## 18. MTG2で追加された個別戦略

### まっすーへのアプローチ（②癖強自己紹介）
- 聞く力はめちゃくちゃ才能ある → そこは自然に褒める
- 自分を覚えてもらう行動が足りない
- **間接的手法**: 他の人（例: れいちゃん）の良い自己紹介を承認 → まっすーに気づかせる
- それでも改善がなければ → 最後のディスカッションで直接伝える
- 例: 「意外性を持ってたらもっと良かったよね」「自分のことじゃなくてもいい、友達の変な話でもいい」

### れいちゃんへのアプローチ（④写真パズル）
- 前回の地図ワークで「もっとリーダーシップ取った方がいい」と言われた
- 今回覚えていて行動した → 「覚えてて動いてたね」と承認
- 覚えてなかった → 「毎回来る必要性もないじゃん。前回で前回だけの人って言ったんだから」（だいが感情ベースで）
- 聞く課題（紙に書けてない等） → りゅうが最後の回収で指摘

### あすかへのアプローチ（全体通し）
- スタンスのブレを一貫して見る（りゅうのメインタスク）
- 合宿前に「今日はこういう感じでやります」と共有するはず → そことの比較
- ブレたら「変わったなら変わったって言ってくれないと」と伝える
- 前回はプレイヤーになっちゃって落ちた → コンサル的スタンスを維持できるか
