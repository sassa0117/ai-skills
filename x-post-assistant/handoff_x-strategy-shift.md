## 戦略転換（言語化完了）

### 「エンジョイ勢」ポジショニング
- 現状のバズポストは全部「稼ぐ文脈」→ 情報商材・転売ヤーと同じ土俵
- 目指す方向: 「エンジョイ勢」文脈への転換
- 参考の空気感: 積読チャンネル、ゆる言語学ラジオ

### 共有おじさんとの違い = 「目」
- 共有おじさん = 「これ見つけた！」→ ふーん、で終わり
- さっさの場合、中古市場を読む目（シンオウ理論）が体に染みついてるから「見えてるもの」が違う
- 「稼ぐ文脈」のフックは結果（金額）。「エンジョイ勢」のフックは発見（目の良さ）
- ガチの知識・眼力が「楽しんでる姿」からにじみ出る構造

### X + note の全体設計
- **Xは集客装置、noteで稼ぐ。比率はnote収益化に寄せる**
- X（エンジョイ勢ポスト）→ フォロー → noteへ
- note無料記事 → 中古市場の面白さ深掘り → 有料へ
- note有料記事/メンバーシップ → 収益の柱

### noteコンテンツ設計（エンジョイ勢トーン）
| 層 | 内容 | トーン |
|---|---|---|
| 無料記事 | 発見系（ブックオフで見つけたやばいもの等） | 面白がってる。稼ぐとは言わない |
| 有料記事 | なぜ気づけたのか。目の裏側にある構造 | エンジョイ勢のまま深さが出る |
| メンバーシップ | リアルタイムの発見共有、仕入れ裏側、コミュニティ | 「一緒に宝探し」感覚 |

### 稼ぐ文脈との決定的な違い
- 稼ぐ文脈 → 「知れば稼げます」（先生と生徒）
- エンジョイ勢 → 「面白くない？なんでこうなるか知ってる？」（同じ目線）
