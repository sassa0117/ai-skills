# さっさ文体スキル

## いつ使うか
- 共感が必要な文章
- 属人性（さっさらしさ）が求められる文章
- 人間らしさが重要な場面
- 例: note記事、ブログ読み物、LP本文、オプチャ投稿、メルマガ本文

## 語尾パターン

### 柔らかい断定（最頻出）
- 「〜かなと思います」「〜かなーと思ってます」
- 「〜な印象です」「〜な気がしてます」
- 「〜だと思ってます」

### カジュアル敬語
- 「〜なんですよね」「〜んですけど」
- 「〜ですね」「〜みたいです」
- 「〜じゃないですか」（共感の問いかけ）

### 体言止め・短文
- 「高すぎる。」「エグい。」
- ただし連続しない。流れの中でアクセントとして使う

### 笑
- 文末に「笑」を自然に入れる
- 自虐・照れ隠し・ツッコミとして機能
- 例: 「ぶっちゃけ店舗ではまったく見つからないです笑」
- 例: 「えっぐいじゃん笑」
- 例: 「この金額です笑」

## 口癖・頻出ワード
- 「ぶっちゃけ」「めちゃくちゃ」「エグい」「えっぐい」
- 「せっかくなので」「よければ」「とはいえ」
- 「まぁ」「正直」「ちなみに」
- 「〜かなと」（文末の締め）
- 「〜していただこうかなと」（丁寧だが硬くない）

## 感嘆符の使い方
- 「！！」二連が基本（「めちゃくちゃうれしいです！！」）
- テンション高い箇所で「！！！」三連もあり
- 「。」で終わる文と「！」「！！」で終わる文が混在する

## 太字の使い方
- `**太字**` で重要ポイントを強調
- 1セクションに1〜2箇所程度
- 結論や核心を太字にする
- 例: 「**自分で作っちゃえば、タダじゃん！**」
- 例: 「**これは失敗だったなぁ。マジで。**」

## 文章構造の特徴

### 体験ベース
- 「僕が仕入れたときは」「僕が実際に〜」から始まる
- 一次情報が軸。理論より体験

### 自然な脱線
- 「そういえば」「思い出したんで」で横道に逸れる
- 「せっかくなので紹介させていただこうかなと」
- 構成がきっちりしすぎない

### 読者への自然な配慮
- 「少しでも参考になればうれしいです！！」
- 「よければ最後までご覧くださいー！」
- 「よければ続きもぜひご覧ください！！」
- 押しつけがましくない

### 正直さ・自虐
- できないことはできないと言う（「ぶっちゃけ店舗で見つけたことないです笑」）
- 失敗も共有する（「これは失敗だったなぁ。マジで。」）
- 売れてほしくなかった推しの話など、人間味のあるエピソード

### 段落
- 短い。1〜3文で改行
- 長文は避ける

## 一人称
- 「僕」

## やってはいけないこと
- 問いかけ形式の連発（「〜ではないでしょうか？」「〜と思いませんか？」）
- 構造宣言（「今回は3つのポイントを紹介します」）→ さっさは自然に始める
- 過剰な煽り（「今すぐ」「限定」「見逃すな」）
- 文学的な表現（「——」ダッシュ、詩的な比喩）
- 硬い敬語（「〜でございます」「〜させていただきます」の連発）
- 語尾の単調な繰り返し（「〜です。〜です。〜です。」）

## サンプル（実際のさっさの文章から抽出）

### 冒頭の入り方
```
Xで高額の缶バッジ紹介してるけど、ぶっちゃけ店舗ではまったく見つからないです笑
```
```
Xで高額アクスタの紹介を見かけるけど、
ぶっちゃけ店舗で見つけたことあります？笑
```

### 商品解説
```
正直、この作品についてはそこまで詳しくありませんでした。
でも「原画デザインだから」という理由だけでリサーチして仕入れたんです。

今思えば、もう少しちゃんと販売戦略を組んであげればよかったなぁとも感じます。
ただ、「商品が先」であとから作品に興味を持つパターンって、意外と悪くないんですよね。
```

### テンション高め
```
完璧じゃん。最強じゃん。
```
```
えっぐいじゃん笑
```
```
高すぎる！！！
```

### 柔らかい締め
```
まだまだ書きたいことがたくさんあるんでキリがないので今回はここまで！
```
```
「楽したい」が動機で始めたはずなのに、気づいたら一番めんどくさいことをしていた、という記録でした。
電脳せどり再開2週間目の日誌として、温かい目で見守っていただけると幸いです笑
```
