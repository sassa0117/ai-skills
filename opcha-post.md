# オプチャ投稿スキル

LINEオープンチャット向けの投稿文を作成するスキル。
さっさ文体スキル（/sassa-writing）と併用前提。

## フォーマットルール

### 改行・空行
- **1文1行**が基本。句点「。」のあとは改行
- 長い文は意味の切れ目で改行して分ける
- **段落の切り替わり（話題転換）は空行2つ**（空行→空行で視覚的に区切る）
- 同じ話題内の文と文の間は空行1つ
- スマホで読む前提。横に長い行を作らない

### URLの扱い
- URLは必ず**単独の行**に置く
- URLの前に「▼ ラベル名」を1行入れる
- URL前後に空行を入れてタップしやすく

```
▼ プラン比較
https://sedorisassa.com/?page_id=5258
```

### 連絡先ブロック（末尾固定テンプレ）
- 「→」でつなげずURLは改行で分離（タップしやすさ優先）

```
質問あればこちらから

📩 公式LINE
https://lin.ee/rBzkuIB

📩 匿名で質問
https://sedorisassa.com/5mvl
```

### リスト
- 「・」を使う（「-」は使わない）
- 各項目は1行で収める
- リストの前後に空行

```
・店舗せどりしかやらない人
・電脳に興味ない人
・そもそもリサーチ自体やってない人
```

## ファイル構造

```
【日付 時間帯 オプチャ投稿用 — テーマ概要】

---ここからコピペ---

（本文）

---ここまで---

【メモ】
- テーマ: ○○
- 投稿時間: ○○
- 使用したs-kenスキル: ○○
```

- 「---ここからコピペ---」〜「---ここまで---」の中身だけがコピペ対象
- 【メモ】は自分用の運用メモ。投稿には含めない

## 文体ルール（/sassa-writing 準拠）

- 一人称「僕」
- 柔らかい断定（「〜かなと」「〜な印象です」）
- カジュアル敬語（「〜なんですよね」「〜じゃないですか」）
- 「笑」は自虐・照れ隠しで自然に
- 「ぶっちゃけ」「めちゃくちゃ」「正直」は使ってOK
- 煽らない。「今すぐ」「限定」「見逃すな」はNG
- 「合う人だけどうぞ」のスタンスを崩さない

## やってはいけないこと

- 問いかけの連発（「〜ではないでしょうか？」「〜と思いませんか？」）
- 構造宣言（「今回は3つのポイントを〜」）
- 過剰な装飾（太字・見出し・区切り線 → オプチャではプレーンテキストのみ）
- 1文が2行以上にまたがる長文
- URLを文中に埋め込む（必ず単独行）
- 「松竹梅」のプラン呼称（フルサポート/スタンダード/ライト）

## 投稿設計のフレームワーク（/sken-step-mail 連携）

投稿ごとにs-kenの5つの教育のどれを使うか意識する:

| 教育 | 使い方の例 |
|------|-----------|
| 信頼 | 正直に合わない人を伝える、自虐、失敗談 |
| 投資 | 時間コスト計算、お金で時間を買う判断 |
| 行動 | LPクリック誘導、「見てみてください」 |
| 手段 | ツールの具体的な使い方・ベネフィット |
| 目的 | リサーチ時間を減らして何ができるか |

## サンプル（良い例）

```
電脳リサーチ、
1日何時間かけてますか？


僕は前まで毎日2〜3時間、
5サイト巡回してました。

セカスト見て、オフモール見て、
まんだらけ見て…

で、結局なにも仕入れられない日もある笑


月に60〜90時間。

時給1,000円で計算しても
6万〜9万円分の時間です。
```

## サンプル（悪い例）

```
電脳リサーチ、1日何時間かけてますか？僕は前まで毎日2〜3時間、5サイト巡回してました。セカスト見て、オフモール見て、まんだらけ見て…で、結局なにも仕入れられない日もある笑　月に60〜90時間。時給1,000円で計算しても6万〜9万円分の時間です。
```

↑ 改行なし・空行なしで壁になっている。スマホで読めない。
